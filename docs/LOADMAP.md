# 実装ロードマップ

フェーズ 4（全ボス実装）の進め方をまとめたドキュメント。  
各ステップは「設計 → 実装 → テストプレイ」のサイクルで進める。

> **進捗**: ステップ 0〜7 はすべて実装完了（PR #15 時点）。以後はフェーズ 5（難易度調整）とビジュアル実装（世界観は [WORLD_DESIGN.md](WORLD_DESIGN.md) 参照）に進む。  
> ※ 本ドキュメントの名称は新世界観（機械王・オイルスライム・キング等）に統一しているが、現行実装のゲーム内テキストは旧名称のまま。

## ステップ 0 — コアシステム整備

フロア固有コンテンツの土台となる共通システムを実装する。

### 設計
- バトルシステム（ターン制、ダメージ計算、勝敗判定）
- NPC セリフシステム（会話ウィンドウ、メッセージ送り）
- フロア遷移システム（階段・扉でのシーン切り替え）
- セーブ・ロードの最終仕様確認（SAVE_FORMAT.md と照合）

### 実装
- `BattleScene` の基本フロー
- NPC 会話 UI
- フロア間遷移ロジック
- セーブ・ロード共通処理（`src/save/`）

### テストプレイ観点
- バトル開始 → 勝利 / 敗北の一通りが動作するか
- NPC に話しかけて会話が完了するか
- セーブ → ロードでゲーム状態が復元されるか

**ブランチ例:** `feature/core-systems`

## ステップ 2 — 2F（uint8 / 16 進数の基本）

### 設計（確定）

**ボス：オイルスライム・キング**
- HP: 40 / 攻撃力: 3
- HP ≤ 10 で 1 回だけ全回復（HP 40 に戻る）
- 勇者と衝突でバトル開始（セリフなし）
- バトル開始時に HP > 255 を検出 → 発覚セリフ → ゲームオーバー（全ステートリセット）

**斬撃威力改定：5 → 10**

**2F マップ：16×16タイル（5×5 カメラ）**
- 入口：下端中央（1F からの階段）
- NPC1（入口付近）：「ここのボスは粘り強いらしいぞ！頑張れよ！」
- NPC2（中盤）：「おまえそんな HP で大丈夫か？」
- オイルスライム・キング：マップ中央上部、直進ルートを塞ぐ
- NPC3（奥の部屋・側道経由でボス前に到達可能）：「ボスに勝てない？もうセーブデータをいじるしか無いよな？どこをいじればいいかって？それは自分で考えろ！」
- 3F 階段：最上部（オイルスライム・キング撃破後に使用可能）

### 実装
- canvas: 320×320px、タイルサイズ: 64px（5×5 タイル表示）
- カメラ追従システム（全フロア共通）
- 2F マップ（壁・通路・部屋のレイアウト）
- オイルスライム・キングの衝突バトル起動
- ボスヒールロジック（HP ≤ 10 で 1 回全回復）
- HP > 255 チート発覚 → ゲームオーバー
- 3 NPC 配置とセリフ
- 3F 遷移（オイルスライム・キング撃破後に解放）
- skills.ts の斬撃威力 5 → 10

### テストプレイ観点
- 通常 HP(10) で挑んで敗北するか（ヒール後に力尽きる展開になるか）
- HP をセーブデータで増やして勝利できるか（HP ≥ 16 で理論上可）
- HP > 255 のとき発覚演出が発動しゲームオーバーになるか
- 側道から NPC3 の部屋へボス前に到達できるか
- オイルスライム・キング撃破後に 3F 階段が使えるか

**ブランチ:** `feature/stage2-boss1`

## ステップ 7 — -1F（裏ボス / float32）※1F のからくり像から常時挑戦可能

### 設計
- -1F の進入経路（1F のからくり像を動かして現れる隠し階段。常時開放だが、勝利には機械王撃破で解放されるメギドが必須）
- -1F ステージレイアウト
- -1F スキルの仕様（機械王から入手した技。float32 として操作）
- 裏ボスのスプライト・ステータス・セリフ
- -1F NPC のセリフ（float32 のビット表現へのヒント）

### 実装
- -1F マップと解放ロジック
- スキル威力の float32 保存・読み込み
- スキルオブジェクトの配置
- 裏ボスの戦闘ロジック
- -1F NPC

### テストプレイ観点
- 機械王クリア前でも -1F に入れるが、カオスには勝てないか（メギドなしでは再生に届かない）
- float32 の正しい解釈でのみ必要な威力を出せるか
- 裏ボスを倒して特別エンディングに到達できるか

**ブランチ例:** `feature/secret-boss`
