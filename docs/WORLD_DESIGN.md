# 世界観設計ドキュメント — 大機関塔

スチームパンク世界観の設定バイブル。キャラクター・フィールド・UI のビジュアルとテキストはすべて本ドキュメントに従う。

## コアコンセプト

蒸気と歯車で動く巨大な自動機械の塔「**大機関塔（だいきかんとう）**」が舞台。
ドラクエ的な王道 RPG（勇者・姫・魔王）の骨格に、機械の魔物が徘徊するスチームパンクの意匠を重ねる。

最大の特徴は**塔の垂直軸が「計算の歴史」になっている**こと。

```
最上階  中枢機関室（電子・シアン発光のサイバー空間）
  ↑
4F-5F   計算機関区画（歯車式計算機・真鍮）
  ↑
2F-3F   工場・蒸気管区画（オイル・銅・錆）
  ↑
1F      機関室入口（レンガ・ガス灯・アナログ）
```

下層は蒸気・歯車のアナログ世界、登るほど電子・発光のデジタル世界に近づく。
ドラクエ感（下層）とサイバー感（上層）が一本の世界観でつながり、プレイヤーが学ぶ概念（16進数 → エンディアン → チェックサム）の高度化とも同期する。

## ストーリー

大機関塔は、不正を見抜く力を持つ**技師の姫**が設計し、守ってきた。
しかし塔の中枢で自律機関「**機械王**」が暴走し、姫を頂に幽閉。塔の機械たちは魔物と化した。
勇者は塔を登り、機械王を倒して姫を救い出す。

### セーブデータの位置づけ

セーブファイルは勇者の魂を写した「**機関銘板（パンチカード）**」。16進数の刻印列でできており、刻印を書き換えることは機械仕掛けの世界の設計を直接いじる行為にあたる。
※ ゲーム内ではこの設定を直接説明しない（メタ発言を避ける方針を維持）。

## 演出方針

- チート発覚: 蒸気噴出+歯車逆回転+グリッチノイズ（パーティクル+カメラシェイク）
- 環境演出: 蒸気パーティクル、ガス灯の明滅
- 上層に行くほど発光（emissive）表現を増やし、サイバー感を強める

## テキスト規約

- 老技師・機械王など年長者/権威の口調は古文調（「わしは」「そなたの」「じゃ」）を維持
- メタ発言（「ゲーム」「プレイヤー」「ファイル」等）はしない。没入感を維持する
- セーブデータ改ざんの「答え」はゲーム内テキスト・ソースコメントに書かない（ヒントはオフセットの示唆まで）
- 「検算」「銘板」など計算を匂わせる語は終盤（機械王戦）まで温存する
