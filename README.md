# ClearBox

16進数とコンピュータのデータ表現を「セーブデータ改ざん」を通じて体験する教育用 2D ドット絵 RPG。

## ゲーム概要

蒸気と歯車で動く「大機関塔」を舞台に、機械の魔物を倒して姫を救うスチームパンク × 王道 RPG ですが、普通にプレイしても強敵には勝てません。  
**セーブデータ（.txt）を直接書き換える**ことで攻略の糸口が開きます。

授業で学んだ 16 進数・2 進数・メモリの概念を、手を動かして実感するためのゲームです。

## 対象プレイヤー

- 16 進数・2 進数の基礎を授業で学んだ人
- コンピュータがデータをどう管理しているか体験したい人
- Chromebook / ブラウザのみで動作（インストール不要）

## 技術スタック

| 項目 | 採用技術 |
|---|---|
| ゲームエンジン | [Phaser.js 3](https://phaser.io/) |
| 言語 | TypeScript |
| バンドラー | Vite |
| ホスティング | GitHub Pages |

## セーブデータの仕組み

セーブ時にブラウザから `.txt` ファイルをダウンロードします。  
ロード時にそのファイルをアップロードします。  
ファイルの中身は **16 進数テキスト** で、テキストエディタで書き換えられます。

詳細は [docs/SAVE_FORMAT.md](docs/SAVE_FORMAT.md) を参照してください。

## セットアップ（開発者向け）

Node.js 20 以上が必要です。

```bash
npm install
npm run dev      # 開発サーバー起動 (http://localhost:5173)
npm run build    # 本番ビルド
npm run preview  # ビルド結果をローカルで確認
```

## デプロイ

`main` ブランチへのプッシュで GitHub Actions が自動的に GitHub Pages へデプロイします。
